## ¿De qué trata el proyecto?

**Japon Trip** reúne distintas secciones relacionadas con turismo y cultura japonesa:

- **Inicio** con presentación general del viaje.
- **Ciudades** recomendadas para visitar.
- **Templos** y lugares tradicionales.
- **Sakura** y experiencias de temporada.
- **Tradiciones** culturales destacadas.
- **Itinerario** orientativo para organizar el recorrido.

El objetivo es ofrecer una guía visual básica y atractiva para descubrir Japón en un solo sitio web.
